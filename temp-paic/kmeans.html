Segmentação: K-Means¶

A ideia verificar se o K-Means consegue segmentar imagens escaneadas ([0-9]s-[0-9].jpg) e fotografadas ([0-9]f-[0-9]{3}.jpg) de fololos de cultivares de guaranazeiros.

Para isso, usarei bibliotecas como NumPy, Matplotlib, Scikit-learn e Scikit-image...

from skimage import transform, io
from glob import glob

def img_resize(image, msize=500):
    t = msize / max(image.shape)
    n = tuple([int(s * t) for s in image.shape[:2]])
    return transform.resize(image, n, mode='reflect')

imgs = []
for file in sorted(glob("./dataset/*.jpg")):
    imgs.append([file.split('/')[-1], img_resize(io.imread(file))])

Acima, as imagens (fotogradas e escaneadas) so carregas em uma lista, sendo que cada item dessa lista encontra-se no seguinte formato: [str("..."), np.array()], onde a primeira posio armazena o nome do arquivo enquanto que a segunda, a imagem em um numpy array.

Agora, as imagens passaro por um Processamento Digital de Imagens (PDI), onde 3 operaes sero realizadas objetivando a remoo de rudos e suavizao das amostras. Em seguida, cada imagem "ps-pdi" anexada lista imgs. Essas operaes, embora apresentem resultados satisfatrios, so bastante lentas.

from skimage import restoration, filters, exposure

def pdi(image):
    image = restoration.denoise_nl_means(image, h=0.15, multichannel=True)
    image = filters.gaussian(image, sigma=1.6, multichannel=True)
    image = exposure.adjust_gamma(image, gamma=0.5, gain=1)
    return image

for img in imgs:
    img.append(pdi(img[1]))

Essa funo ser til quando formos iterar a lista imgs, executando o K-Means para cada imagem presente, mais a frente.

def recreate_image(codebook, labels, w, h):
    """Recreate the (compressed) image from the code book & labels"""
    d = codebook.shape[1]
    image = np.zeros((w, h, d))
    label_idx = 0
    for i in range(w):
        for j in range(h):
            image[i][j] = codebook[labels[label_idx]]
            label_idx += 1
    return image

from sklearn.cluster import KMeans
def kmeans(image2d, shape, k=3):
    kmeans = KMeans(n_clusters=k, random_state=0).fit(image2d)
    labels = kmeans.predict(image2d)
    return recreate_image(kmeans.cluster_centers_, labels, *shape)

As dimenses de cada imagem sero, agora, convertidas de ($w \leq $ msize, $h \leq $ msize, $d = 3$) para $(w*h, d)$. Sendo que $w$, $h$ e $d$ correspondem a largura, altura e a quantidade de canais de cor (depth), respectivamente.

O resultado desse processo (que aplicado s imagens j pr-processadas) armazenado em img2darray, enviado a funo kmeans() junto das dimenses originais da imagem e, em seguida, anexado a lista imgs.

Thresholding e finalização¶

Agora iremos binarizar os pixels de cada amostra para, em seguida, multiplicarmos o resultado com a imagem original para obtermos apenas a regio da folha, sem o fundo.

Iremos utilizar o algoritmo de Otsu para realizar a binarizao das amostras:

In [8]:
from skimage.color import rgb2gray
from skimage.filters import threshold_otsu

for img in imgs:
    gray = rgb2gray(img[3])
    binary = gray > threshold_otsu(gray)
    img.append(binary)

Conclusões¶

Este experimento utilizou 10 amostras de fololos dos cultivares de guranazeiros, sendo uma metde composta de imagens fotografadas e outra de escaneadas, para testar a eficcia de segmentao do algoritmo de classificao no supervionada (clusterizador) K-Means.

Ao longo dos testes, observou-se que a qualidade da segmentao dos fololos foi consideravelmente dependente das tcnicas de PDI utilizadas, principalmente quando se tratando de imagens fotografadas. Isso, de certa forma, um problema que inviabiliza a utilizao deste algoritmo, especialmente para a segmentao dessa categoria de imagens (no contexto deste experimento; i.e., imagens fotografadas), uma vez que no se obteve resultados majoritariamente satisfatrios.

Curiosamente, para as imagens escaneadas, o algoritmo no teve problemas (quando comparado com imagens fotografadas e utilizando os mesmos parmetros) visivelmente negativos segmentao dos fololos. Muito provavelmente, tal resultado esteja atrelado, dentre outros fatores, uniformidade de brilho dessas imagens e a ausncia de sombras passveis imagens fotografadas.

Muito em breve, espera-se utilizar outras tcnicas de segmentao no supervisonadas como, por exemplo, a Watershed ou o algoritmo Randon Walker, para essas mesmas amostras a fim de comparar seus desempenhos com o K-Means.
